linux中时钟同步问题，ntp同步时钟

2019-07-02
linux, ntp

问题

在linux系统运行过程中很容易造成时钟错乱的现象，如果不经常同步那么过一段时间就会发现服务器的时钟和标准时间差异性很大了，这个时候就需要将自身的时钟和服务器的时钟进行同步。这里采用ntp的形式进行同步。

查看时钟状态

首先查看时钟是否同步的，通过命令timaedatectl

NTP enabled表示是否开启ntp同步功能，这里默认一般都是开启的如果没有开启则可通过命令timedatectl set-ntp yes来进行开启，开启并不代表就同步了，还需要安装ntp
NTP synchronized表示当前状态的时间是否和时钟服务器的时间同步的，如果是yes表示已经和时钟服务器的时钟同步了。因ntp是平滑同步改变时间，在初次安装运行ntp时该状态会为no，在时钟和服务器同步了后就会变成yes

安装配置ntp

通过配置/etc/ntp.conf文件来配置ntp的信息server属性表示要同步的标准时钟服务器，ntp会去和这些服务器进行时钟同步。
输入systemctl start ntpd来启动ntp服务，输入systemctl enable ntpd来开机启动ntp服务
