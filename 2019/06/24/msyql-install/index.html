启动mysql

mysql启动之前会初始化数据库，如果/etc/my.cnf配置文件中的datadir有值或不是默认，那么走默认初始化就会出错，这时手动初始化数据库即可,在初始化时指定datadir

注意: 在mysql初始化数据库是会根据配置信息或默认信息去初始化一些属性，这些属性只能在初始化时进行应用，当数据库初始化完成后，这些属性将不能被更改，如: 忽略大小写： lower_case_table_names=1

用户相关

mysql和oracle在用户和数据库的管理上有很大差别，oracle是用户下面存在表、表空间、数据等，而mysql是root用户创建一个database，将database的权限赋予用户，用户再来使用表、数据等。
