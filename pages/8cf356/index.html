docker常用命令

# 常用命令

  1. 查看当前运行的所有容器
 docker ps -a
1

# 日志相关

docker logs container-id
1

# 推送镜像

登录仓库

docker login 仓库地址
1

打标签

docker tag 本地的image名称 仓库地址/项目路径/推送到仓库的镜像名称:版本号
1

开始推送

docker push 仓库地址/项目路径/推送到仓库的镜像名称:版本号
1

# 删除相关

  1. 停止所有容器（container），这样才能够删除其中的images：
docker stop $(docker ps -a -q)
1
  1. 如果想要删除所有容器（container）的话再加一个指令：
docker rm $(docker ps -a -q)
1
  1. 查看当前有那些镜像（images）
docker images
1
  1. 删除镜像（image）
docker rmi (image id)
1
  1. 想要删除镜像（images）id为None的image的话可以用
docker rmi $(docker images | grep None | awk '{print $3}')
1
  1. 要删除全部镜像（images）的话
docker rmi $(docker images -q)
1

# 容器相关

查看容器信息

docker inspect name
1

查看容器日志

docker logs name
1

查看容器资源使用

docker stats name
1

执行容器内部命令

docker exec name ls /
1

进入容器内部

docker exec -it name /bin/bash
1

# 网络相关

查看现有网络类型

docker network ls
1

创建网络类型

docker network create -d 类型（overlay network/bridge network） 网络空间名称
